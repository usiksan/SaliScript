Данный проект представляет собой скриптовый язык reduced c++ (rc++)
и содержит компилятор со скриптового языка, генератор
байт-кода, описание байт-кода и виртуальную машину для исполнения
байт-кода

Проекты
SvVMachine.pro тестовый проект для проверки компиляции виртуальной машины
SvCompiler.pro тестовый проект для проверки компиляции и генерации байт-кода со скриптового языка rc++
